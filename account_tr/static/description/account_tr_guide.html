User Guide: account_tr Module

1. Fiscal Positions

The fiscal positions functionality allows you to manage and define various fiscal positions for your accounting needs. You can create, edit, and view fiscal positions using the provided interface.

Fields:

  • Code: An identifier for the fiscal position.
  • Code2: An additional code field.
  • Code3: Another additional code field.

2. Fiscal Position Categories

Manage categories for fiscal positions to better organize and classify them.

3. Accounting Journals

Interface for managing and defining accounting journals.

4. Taxes

Manage different types of taxes and their related configurations.

5. Banks

Interface for defining and managing bank details.

6. Company Configurations

Manage and configure company-related details and settings.
